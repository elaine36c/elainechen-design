The app has been designed to be intuitive, easy to navigate and simple to use.


Iterated and Tested.

The design proven by testing and fit to our user's specifications, having been developed and iterated through multiple rounds of testing. User testing revealed elements that disrupted the flow of navigation. We were able to iterate based on feedback and improve these aspects.

User testing methods used to research the experience include:

  • Think Aloud
  • Cognitive Walkthroughs
  • Interviews
  • A/B Testing
  • Surveys & Questionnaires

Refined and Redesigned.

  • Checklist became a scrollable dropdown list to reduce clutter and minimise confusion.
  • Original arrow indicator wasn't intuitive so this was changed to a recognisable drop down arrow icon.
  • Browse button added to allow users to quickly navigate back to add more items to their plan.

Simple Day Plan.

  • The list schedule became drag and drop for easy customisation.
  • Edit icon was made more clear with text and colour added.
  • The minus button was changed to be accessible with ‘edit’ and a drop-down menu.
  • Icons for activities were added for easy identification and distinction.

Clear Categories.

  • The list categories colour scheme was changed to blue. The icons were made smaller with a white background for a more professional finish. The test was made more clear.

Intuitive Browsing.

  • Browse all categories was added.
  • Photos changed into a scrolling card format.
  • The categories were changed to a top nav list with ‘browse all’ as the default page.
