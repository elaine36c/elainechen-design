AuTour

Introduction

The brief for this project was to research and design an interactive user interface for an autonomous vehicle. In order to identify the problem areas and user needs to address, we had to consider the different industries or fields that could be affected. Our group chose to focus on the future of tourism and tour transport as our context. Through this scope we created: AuTour.

The User Interface

AuTour is an autonomous planning application and user interface that works in conjunction with an autonomous tour vehicle. Its aim is to provide users with a memorable, personalised and efficient tour experience.

Our final interface design was fit to our target user's specifications. It has been developed and iterated through multiple rounds of testing. User testing revealed elements that disrupted the flow of navigation. We were able to iterate based on feedback and improve these aspects.

User testing methods used to research the experience include:

  • Think Aloud
  • Cognitive Walkthroughs
  • Interviews
  • A/B Testing
  • Surveys & Questionnaires


The design began from initial sketches, paper prototypes and wireframes. Each iteration was thoroughly tested with recruited users. The feedback from each of these testing sessions were used to redesign and develop the interface to address all of their needs, problems and frustrations. This lead to multiple reiterations of concepts, features and aesthetics.
